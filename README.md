## About The Project

![portfolio-picture](https://i.imgur.com/l7hmGbA.png)

This is my React based portfolio website, which displays a few projects of mine. I found my [old portfolio website](https://jonwaaler.github.io/) was more difficult to update content and maintain a clean code base. I also disliked the layout, light theme and wanted to use React for readability. Along with a json file to store content of each project I want to display.

New Portfolio Website Features :
* Dark theme with blue neon lights and relaxing snowy effects.
* Uses React, making it easy to maintain and organize code in components.
* Uses `projects.json` file to store project information all in a single location.

### Installation

Clone the repository then install npm dependencies

1. Clone the repo
   ```sh
   git clone https://github.com/JonWaaler/react-portfolio.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```

## Usage

1. Start the project locally by using the react script 'start'
`npm start`
the project will open a webpage automatically.

2. To update project data go to: `src/projects.json`
